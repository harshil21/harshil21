## Hi there 👋

I'm Harshil, an aerospace engineer with a deep passion for creating useful and well-crafted software. Outside of engineering, I'm a maintainer of [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot), one of the most popular Python libraries for building Telegram bots.

Most of the code I ship is hand-written or, at the very least, carefully reviewed for correctness if done by AI.

### 🚀 Projects I've contributed significantly to

- [**python-telegram-bot**](https://github.com/python-telegram-bot/python-telegram-bot) — A pure Python, asynchronous interface for the Telegram Bot API
- [**AirbrakesV2**](https://github.com/NCSU-High-Powered-Rocketry-Club/AirbrakesV2) — Airbrakes control system for high-powered rockets
- [**FIRM**](https://github.com/NCSU-High-Powered-Rocketry-Club/FIRM) — Filtered Inertial Rotation Module (flight computer firmware)
- [**TGDates**](https://github.com/harshil21/TGDates) — A simple, extensible date and time picker as a Telegram Web App
- [**Aloft**](https://github.com/NCSU-High-Powered-Rocketry-Club/Aloft) — Ground station software for high-powered rocketry
- [**carbonpy**](https://github.com/harshil21/carbonpy) — An organic chemistry module for Python
